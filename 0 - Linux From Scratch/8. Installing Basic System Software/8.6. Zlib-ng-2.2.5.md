The Zlib-ng package contains compression and decompression routines used by some programs.

---
## 8.6.1. Installation of Zlib-ng

Prepare Zlib-ng for compilation:

```shell
./configure --prefix=/usr --zlib-compat
```

Compile the package:

```shell
make
```

To test the results, issue:

```shell
make check
```

Install the package:

```shell
make install
```

Remove a useless static library:

```shell
rm -fv /usr/lib/libz.a
```
